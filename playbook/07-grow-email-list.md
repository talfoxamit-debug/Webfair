# Grow the Email List

You have **47 clean emails** ready to send (`instantly-fence-leads.csv`). Here's
how to add more, fastest and cheapest first.

## Where the 254 leads stand
- **47**, verified emails, already in your Instantly campaign ✅
- **52**, have a real business website but no *published* email → **guess + verify** (below)
- **50**, no website at all → **phone only** (CRM 🔥 No-website filter + call scripts)
- The rest were directory-listing "sites" (no real domain/mailbox), skip for email.

---

## Tactic 1: `info@` guess + verify (free, do this first)
Most small contractors use `info@theirdomain.com`. I've generated the guess for
all 52 site-but-no-email leads: **`guessed-emails-to-verify.csv`**.

1. Log into **MillionVerifier** (you already use it).
2. Upload `guessed-emails-to-verify.csv` → run bulk verification.
3. Keep **only** the rows marked **"OK / Valid / Deliverable."** Delete everything
   else (catch-all, risky, invalid), sending to those bounces and hurts your domain.
4. The file is already in your Instantly format (email, company_name, city,
   state, phone, website, **icebreaker**), so the survivors upload straight into
   the same campaign and sequence. Map the columns the same way as before.

Expect roughly **30–50%** to come back valid, that's ~15–25 more real leads for
about $0.

> ⚠️ Never send to a guessed email you haven't verified. One batch of bounces on a
> warmed inbox can undo weeks of warmup.

## Tactic 2: Apollo / Outscraper / Hunter (deeper, uses your account)
For the ones that don't verify, a data provider can often still find a real
mailbox. Use **`enrich-domains-apollo.csv`** (company + domain list):

- **Apollo.io** (free tier): Companies → import/search by domain → reveal emails → export.
- **Outscraper** "Emails & Contacts Scraper": paste the domains → returns emails found on the web.
- **Hunter.io** Bulk Domain Search (25 free/mo): upload domains → returns every email on each domain.

Then run whatever they return back through MillionVerifier before adding to Instantly.

## Tactic 3: Work the phone-only 50
The 50 no-website leads have **no** email anywhere, but they're your **hottest**
prospects (no site = easiest pitch). These are phone jobs: CRM → **🔥 No-website
only** filter → call scripts (playbook/02). That's where the fastest first wins are.

---

**Net:** send the 47 now, verify the 52 guesses this week (+~20), and keep the
Instantly ramp going while you work the phone-only list by hand.
